
# gb.setFrameRate() for Gamebuino Classic

[Back to Home](./../../../README.MD) | [Back to Classic Summary](./README.MD)

## Description

Changes how many times per second the program runs.

## Syntax

```
gb.setFrameRate(fps);
```

## Parameters

fps (byte): default value is 20 updates/frames per second.

## Returns

None
