
# gb.changeGame() for Gamebuino Classic

[Back to Home](./../../../README.MD) | [Back to Classic Summary](./README.MD)

## Description

Flashes LOADER.HEX from the micro SD card so you can select and flash another game/program.

## Syntax

```
gb.changeGame();
```

## Parameters

None

## Returns

None
